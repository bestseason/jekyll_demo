欢迎来到我的博客!

Feb 1, 2016

经过两天的努力，我的博客终于成功在Github安家。

简单记录一下整个博客的搭建过程：

首先安装jekyll，jekyll需要以下的包：

  • Ruby & Ruby Development Kit : 可以从这里下载
  • jekyll
  • Python : 推荐使用2.7.5版本. 可以从这里下载
  • Pygments

1.从http://rubyinstaller.org/downloads/安装Ruby，然后安装到你想安装的目录，例如：C:\Ruby

2.下载符合Windows架构和Ruby版本的“DEVELOPMENT KIT”installer。例如，DevKit-mingw64-64-4.7.2-20130224-1432-sfx.exe是用于64位Windows系统的Ruby 2.0.0 x64。从上面所说的地址安装Ruby development kit然后解压它到目录，例如c:\devkit

运行如下命令

ruby dk.rb init

生成接下来要用的config.yml

3.编辑生成的config.yml，使之包含已经安装的Rubies。它可能看起来像这样

#This configuration file contains the absolute path locations of all
#installed Rubies to be enhanced to work with the DevKit. This config
# file is generated by the ‘ruby dk.rb init’ step and may be modified
# before running the ‘ruby dk.rb install’ step. To include any installed
# Rubies that were not automagically discovered, simply add a line below
# the triple hyphens with the absolute path to the Ruby root directory.
#
# Example
#
# ---
# - C:/ruby19trunk
# - C:/ruby192dev
#
---
- C:/ruby

4.运行如下命令安装DevKit。这一步会安装（或更新）一个operating_system.rb文件到相关目录。

ruby dk.rb install

5.用以下命令安装Jekyll

gem install jekyll

6.现在可以运行Jekyll了

jekyll new myblog
cd myblog
jekyll serve

访问http://localhost:4000
